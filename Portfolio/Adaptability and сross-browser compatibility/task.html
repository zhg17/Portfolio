Description

Finally, let's focus on the adaptability and cross-browser compatibility of your website.

At the moment, the navigation bar of your site is displayed okay on computers and laptops, but it does not look great on tablets and phones. Let's make a hamburger menu for those visitors who are using mobile devices. The hamburger menu refers to a type of menu that unfolds after clicking the button with three bars:

Such appearance does remind you of a hamburger a little, right? You've probably seen menus of this type a hundred times before.

Don't forget to test how your menu is displayed on devices with different screen sizes. At the same time, make sure that the menu and other elements of your site display well on different devices and browsers.

Objectives
  1. In the HTML file, create an element with an icon that will open the navigation menu when pressed on mobile devices.
  2. Use CSS to make this icon appear on your site when you view it on devices with smaller screen sizes. The @media rule and the display or visibility property will help you do this.
  3. Create the menu itself. Set the hamburger class to the tag that wraps the menu elements.
  4. Specify the properties for the menu so that it is located where you want it.
  5. Write a script so that when you click the menu icon, it opens and the icon appearance changes (see the examples).
  6. Complete your program so that the menu closes when you click the icon.
  7. Make sure that other elements of your site also display okay on different devices and in different browsers. If necessary, adjust your code.
